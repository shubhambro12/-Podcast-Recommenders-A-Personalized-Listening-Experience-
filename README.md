# -Podcast-Recommenders-A-Personalized-Listening-Experience-
A podcast recommender is a system designed to suggest podcasts to users based on 
their listening habits, preferences, and trends. 
It utilizes algorithms to tailor recommendations.
Developed an interactive AI chat assistant, podcast search, and detailed podcast info pages to enhance discovery. 
Implemented chat history and a clean, responsive UI for a seamless user experience
